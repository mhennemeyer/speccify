Features:

  • Nested Contexts
  • Custom matchers are no-brainers with def_matcher()
  • Blazing Fast
  • 100% Ruby 1.9 and 1.8 compatible
  • Rails out of the box!
  • Sophisticated Expectations/Matchers System
  • < 300 LOC

Install Speccify:

sudo gem install speccify

Using Speccify

Create a testfile:

        
# test_object.rb  

       require "rubygems"  
       require "speccify"  

       describe Object do  
         before do  
           @obj = Object.new  
         end 

         it "is not nil" do
           @obj.should_not be_nil
         end

         it "can be frozen" do
           @obj.freeze
           @obj.should be_frozen
         end
       end

Run it with the ruby command:

$ ruby test_object.rb
Loaded suite -
Started
..
Finished in 0.001820 seconds.

2 tests, 2 assertions, 0 failures, 0 errors
